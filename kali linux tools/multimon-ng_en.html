Multimon-NG Package Description

MultimonNG a fork of multimon. It decodes the following digital transmission modes:

  • POCSAG512 POCSAG1200 POCSAG2400
  • EAS
  • UFSK1200 CLIPFSK AFSK1200 AFSK2400 AFSK2400_2 AFSK2400_3
  • HAPN4800
  • FSK9600
  • DTMF
  • ZVEI1 ZVEI2 ZVEI3 DZVEI PZVEI
  • EEA EIA CCIR
  • MORSE CW

Source: https://github.com/EliasOenal/multimon-ng

multimon-ng Usage Example

Take raw input from rtl_fm (-t raw), add the POCSAG512, POCSAG1200, POCSAG2400, and SCOPE modules (-a POCSAG512 -a POCSAG1200 -a POCSAG2400 -a SCOPE), decode in alpha mode (-f alpha), reading from stdin (/dev/stdin):

[email protected]:~# rtl_fm -f 149.614M -s 22050 -p -19 | multimon-ng -t raw -a POCSAG512 -a POCSAG1200 -a POCSAG2400 -a SCOPE -f alpha /dev/stdin
